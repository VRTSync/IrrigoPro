#92 — Live Billing Exposure Totals on Billing Manager Dashboard
What & Why

The current Billing Manager dashboard only reflects billable (approved) work and does not provide visibility into unapproved tickets that are still in the irrigation manager review stage. This limits the ability to understand how the month is progressing in real time.

We need to introduce a live billing exposure layer that shows both approved and unapproved financial totals without allowing premature billing or editing. This preserves the approval-gated workflow while giving the billing manager a complete, real-time financial picture of the current billing period.

The system must clearly distinguish between:

Approved (billable) work
Unapproved (pending manager review) work
Total combined exposure

This is a visibility feature only, not a workflow bypass.

Done looks like
Company-level (top-left dashboard box)

Displays three values:

Approved:    $X,XXX.XX
Unapproved:  $X,XXX.XX
Total:       $X,XXX.XX
Reflects all customers combined
Updates live as tickets are created and approved
Customer cards (left-side list)

Each customer shows:

This Month:

Approved:    $X
Unapproved:  $X
Total:       $X

Optional status indicator:

Up to Date (no work)
Pending Approval (only unapproved)
Ready to Bill (only approved)
Mixed (both present)
Data rules
Approved = tickets in Approved / Passed to Billing AND not yet billed
Unapproved = tickets in Pending Manager Review
Total = Approved + Unapproved
Billed tickets are excluded entirely from these totals
Workflow integrity preserved
Billing manager cannot edit or bill unapproved tickets
Unapproved tickets remain outside billing selection workflows
This is strictly read-only visibility
Out of scope
Changing approval workflow (#91)
Billing lock logic (#89)
Financial calculation logic (#88)
Adding edit access to unapproved tickets
Changing invoice generation behavior
