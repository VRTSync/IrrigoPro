## Part B — Job-type colors (match the customer guide, section 02)

The customer-facing explainer "Understanding Your Invoice," section 02
"Understand the Job Types," teaches customers a fixed color language for
the three job types. The invoice should use the same colors so the two
documents reinforce each other. Today the reconciliation summary renders
every type in the same navy on a gray bar — no per-type color.

| Job type | Reference | Color | Hex |
|---|---|---|---|
| Work Order | WO-#### | Blue | `#1E5A99` (the PDF's existing navy) |
| Billing Sheet | BS-#### | Amber/Orange | `#B06820` |
| Wet Check Billing | WCB-#### | Green | `#5E8C2A` |

Values sampled from the shipped guide's section-02 card title bars
(Work Order `≈#185898` ≈ the PDF navy — reuse `#1E5A99`; Billing Sheet
`≈#B06820`; Wet Check `≈#588828`, use `#5E8C2A` for contrast under white).

**Locked: job-type colors are FIXED constants, not brand-derived.**
`brandColors` varies per company (extracted from the uploaded logo). The
customer guide is one fixed document, so these colors must be identical
on every company's invoice or they won't match the guide. Do NOT wire
them to `brandColors`.

## Tests

Extend the cover-page render test (or add `pdf-cover-header.test.ts`):

```typescript
it("renders the colored brand band with the navy brand color", () => {
  const html = coverPage(vmWith({ brandColors: { ...DEFAULT_BRAND_COLORS, navy: "#123456" } }));
  expect(html).toContain("cover-brand-band");
  expect(html).toContain("background:#123456");
});

it("shows the logo inside the tile when logoDataUri is present", () => {
  const html = coverPage(vmWith({ company: { logoDataUri: "data:image/png;base64,AAA", name: "Acme" } }));
  expect(html).toContain("cover-logo-tile");
  expect(html).toContain('src="data:image/png;base64,AAA"');
  expect(html).not.toContain("cover-logo-tile-empty");
});

it("falls back to the company initial when no logo", () => {
  const html = coverPage(vmWith({ company: { logoDataUri: null, name: "Greenfield" } }));
  expect(html).toContain("cover-logo-tile-empty");
  expect(html).toContain(">G<");
});

it("renders the invoice number in the below-band block", () => {
  const html = coverPage(vmWith({ invoice: { invoiceNumber: "INV-20418" } }));
  expect(html).toContain("INVOICE #INV-20418");
  expect(html).toContain("Billing Period:");
});

// Part B — job-type colors
it("exposes fixed job-type colors matching the guide", () => {
  expect(JOB_TYPE_COLORS.workOrder).toBe('#1E5A99');
  expect(JOB_TYPE_COLORS.billingSheet).toBe('#B06820');
  expect(JOB_TYPE_COLORS.wetCheck).toBe('#5E8C2A');
});

it("applies each job-type color to its recon section + labels", () => {
  const css = buildFullCSS(DEFAULT_BRAND_COLORS);
  expect(css).toContain('.recon-type-wcb { color: #5E8C2A; }');
  expect(css).toContain('#B06820'); // BS amber section bar

it("job-type colors do NOT change with brandColors", () => {
  const css = buildFullCSS({ ...DEFAULT_BRAND_COLORS, navy: '#000000', green: '#000000' });
  expect(css).toContain('#B06820');
  expect(css).toContain('#5E8C2A');
});
```
