## Design Context (locked decisions)

Inherited from WC Manager Experience Slice 0 (Slice 8 section).

**`/billing-workspace` is the billing manager's home** at
`artifacts/irrigopro/src/pages/billing-workspace.tsx` (1405 LOC, Task #709).
Status strip (4 tiles) + queue (filter+sort+pagination) + sticky
filters. This slice's job is to ship a sibling for the irrigation
manager — NOT modify billing-workspace.

**The role lands here by default.** Once Slice 5 (`managerNav`) is
landed, the irrigation manager's role block in App.tsx routes them to
`/manager-workspace` as their default `/`. Until then, they continue
to land on the existing ManagerDashboard.

**Tile semantics for the irrigation manager:**
- Tile 1: "Wet Checks Pending Review" — count of WCs in
  `submitted` + `pending_manager_review` statuses.
- Tile 2: "WOs Awaiting Manager Approval" — count of WOs in
  `pending_manager_review` status.
- Tile 3: "Findings Needing Routing" — count of `wet_check_findings`
  with `techDisposition === "needs_review"` and no `target` set.
- Tile 4: "This Week's Approved" — count of WCs + WOs + BS with
  approvedAt in the current ISO week.

**Queue defaults for the irrigation manager:** show wet checks first
(type=wc), then work orders (type=wo), then billing sheets (type=bs).
Sort by oldest first (`age_desc`).

**Endpoint pattern matches `/billing-workspace`.** The backend gets
`/api/manager-workspace/queue` and `/api/manager-workspace/status-strip`
that mirror the billing-workspace endpoints exactly. Role-gated to
irrigation_manager / company_admin / super_admin.

**(B2) Status-strip endpoint** mirrors
`billing-workspace-routes.ts:647+`:

```typescript
app.get("/api/manager-workspace/status-strip", requireAuthentication, async (req, res) => {
  if (!isManagerAllowed(req)) {
    res.status(403).json({ message: "Access denied." });
    return;
  }
  const [wetChecks, workOrders, findings, billingSheets] = await Promise.all([
    scopedWetChecks(req),
    scopedWorkOrdersForManager(req),
    scopedFindingsNeedingRouting(req),
    scopedBillingSheets(req), // imported from billing-workspace-routes
  ]);

  const now = Date.now();
  const weekAgo = now - 7 * 86_400_000;
  const tsOf = (v: any) => (v ? new Date(v).getTime() : NaN);

  const wcsPendingReview = wetChecks.filter((w) => ACTIVE_WC.has(w.status)).length;
  const wosAwaitingApproval = workOrders.filter((w) => ACTIVE_WO_FOR_MANAGER.has(w.status)).length;
  const findingsNeedingRouting = findings.length;
  const approvedThisWeek =
    wetChecks.filter((w) => APPROVED_WC.has(w.status) && tsOf(w.approvedAt ?? w.updatedAt) >= weekAgo).length +
    workOrders.filter((w) => ["approved", "billed", "invoiced", "completed_approved"].includes(w.status) && tsOf(w.approvedAt ?? w.updatedAt) >= weekAgo).length +
    billingSheets.filter((s) => ["approved", "billed", "invoiced"].includes(s.status) && tsOf(s.approvedAt ?? s.updatedAt) >= weekAgo).length;

  res.json({
    indicators: {
      wcsPendingReview,
      wosAwaitingApproval,
      findingsNeedingRouting,
      approvedThisWeek,
    },
  });
});
```
